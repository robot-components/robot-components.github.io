# RTAC 홈페이지 관리자 기능 가이드

> 작성일: 2026년 4월 3일
> 배포 URL: https://robot-components.github.io/

---

## 관리자 로그인

- 각 페이지 우상단 **"관리자"** 버튼 클릭
- 비밀번호: `2840`
- 로그인 후 황색 배너("✦ 관리자 모드") 표시됨
- 로그아웃: 황색 배너 우측 "로그아웃" 클릭 또는 "관리자 종료" 버튼 클릭

---

## 페이지별 관리자 기능

### 1. 센터 소개

| 기능 | 설명 |
|------|------|
| 헤더·푸터 센터명 수정 | 한글명 / 영문명 변경 |
| 히어로 배너 수정 | 제목, 부제, 뱃지(쉼표 구분), 버튼 텍스트 변경 |
| 센터 소개 본문 수정 | 소개 텍스트 편집 |
| 지원구조 카드 관리 | 카드 추가/수정/삭제, 카드 색상 변경, 항목 추가/삭제 |
| 주요 업무 관리 | 업무 추가/수정/삭제, 소제목·항목 블록 편집, 연결 페이지 지정 |
| 시험평가지원 프로세스 관리 | 단계 추가/수정/삭제 |
| 오시는 길 수정 | 주소, 이메일, 웹사이트, 지도 embed URL 변경 |
| 장비 데이터 내보내기 | 장비 데이터 JSON 복사 (Claude 인수인계용) |

### 3. 보유 장비

| 기능 | 설명 |
|------|------|
| 카테고리 관리 | 카테고리 추가/수정/삭제, 드래그앤드롭 순서 변경 |

#### 주요 사양 입력 형식
```
섹션명:
항목명 | 값
항목명 | 값

섹션명2:
항목명 | 값
```

## 데이터 저장 방식

- 모든 데이터는 브라우저 **localStorage**에 저장됨
- 브라우저/기기가 달라지면 데이터가 공유되지 않음
- 데이터 초기화 시 기본값(DEFAULT_*)으로 복원됨

### localStorage 키 목록

| 키 | 내용 |
|----|------|
| `rtac_site` | 헤더·푸터 센터명 |
| `rtac_hero` | 히어로 배너 |
| `rtac_intro` | 센터 소개 본문 |
| `rtac_support` | 지원구조 카드 |
| `rtac_works` | 주요 업무 |
| `rtac_process` | 시험평가지원 프로세스 |
| `rtac_location` | 오시는 길 |
| `rtac_eq_v2` | 장비 목록 |
| `rtac_eq_cats` | 장비 카테고리 |
| `rtac_kolas` | KOLAS 규격 목록 |
| `rtac_rooms` | 회의실 목록 |
| `rtac_notices` | 공지사항 목록 |
| `rtac_notice_cats` | 공지사항 카테고리 |
| `rtac_faqs` | FAQ 목록 |
| `rtac_faq_cats` | FAQ 카테고리 |
| `rtac_res` | 회의실 예약 내역 |
